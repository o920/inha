- 프로젝트 제목 : omok.py
- 프로젝트 설명 : 인공지능과 오목을 둘 수 있는 프로그램
- 작동 전 준비사항 :
 ⅰ. anaconda prompt를 다운로드 받아야 함 다음 사이트를 통해 받을 수 있음
	https://www.anaconda.com/distribution/ 
 ⅱ. 오목 규칙을 제대로 이해하고 오목을 둘 수 있는 사람. 렌주룰은 적용하지 않음
	http://omok.mgame.com/omok_init/ifrm_rule1.mgame
-작동 방법 :
 ⅰ. 파일을 다운로드 받고, 압축을 푼다.
       파일 내에 들어있는 image 폴더는 실행에 필요한 폴더임으로 코드파일과 가급적이면 같은 폴더에 둔다.
 ⅱ. anaconda prompt를 이용해서 실행시킨다.
	① cd (폴더이름) 명령어를 통해서 해당 코드가 들어있는 폴더로 이동을 한다.
	② python omok.py 를 입력 후 엔터를 입력하여 파일을 실행시킨다.
- menu세부 사항 :
 ⅰ. Undo :  한 수를 무를 수 있다.
 ⅱ. White : 선을 바꿀 수 있다.
 ⅲ. 33Ban : 한 수에 중복된 3을 두개 이상 만드는 것을 금지시킨다.
 ⅳ. New Game : 새로운 게임 시작한다.
 ⅴ. Quit Game : 게임을 종료한다.
